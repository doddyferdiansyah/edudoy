Level: Beginner

Untuk memahami keamanan jaringan, kita perlu menguasai dasar-dasar jaringan komputer. Bayangkan jaringan seperti sistem jalan raya: ada alamat rumah (IP), jalan (protokol), dan aturan lalu lintas (port). Jika kita tidak paham jalan raya, kita tidak bisa menjaga lalu lintas tetap aman.

ğŸ“– Komponen Dasar Jaringan

  • IP Address: alamat unik untuk setiap perangkat di jaringan.
  • Port: pintu komunikasi untuk layanan tertentu (contoh: port 80 untuk HTTP, port 443 untuk HTTPS).
  • Protokol: aturan komunikasi antar perangkat (contoh: TCP, UDP, HTTP, DNS).
  • MAC Address: identitas unik perangkat di level hardware.
  • Router & Switch: perangkat yang mengatur lalu lintas data.

Model OSI membagi komunikasi jaringan menjadi 7 lapisan:

Lapisan Fungsi Contoh
7. Application Interaksi dengan aplikasi pengguna HTTP, FTP, SMTP
6. Presentation Format data, enkripsi, kompresi SSL/TLS
5. Session Mengelola sesi komunikasi NetBIOS, RPC
4. Transport Pengiriman data end-to-end TCP, UDP
3. Network Routing & alamat IP IP, ICMP
2. Data Link Alamat fisik & frame Ethernet, ARP
1. Physical Media fisik & sinyal Kabel, Wi-Fi

ğŸ§  Analogi Dunia Nyata

  • IP Address: seperti alamat rumah di kota.
  • Port: seperti nomor kamar di sebuah hotel.
  • Protokol: seperti bahasa yang digunakan untuk berkomunikasi.
  • Router: seperti polisi lalu lintas yang mengatur arah kendaraan.

Coba eksplorasi jaringan di perangkatmu:

  1. Buka command prompt/terminal.
