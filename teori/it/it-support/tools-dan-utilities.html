🧰 Tools & Utilities

Level: Intermediate

Tools & Utilities adalah perangkat bantu yang digunakan IT Support untuk mendiagnosis, menganalisis, dan memperbaiki masalah. Bayangkan seperti kotak perkakas teknisi: ada obeng, tang, multimeter — di dunia IT, kita punya ping, Task Manager, antivirus, dan lainnya.

📖 Kategori Tools

  • Monitoring & Diagnosis
    • Task Manager (Windows) / Activity Monitor (Mac).
    • Resource Monitor, Performance Monitor.
  • Network Utilities
    • ping, tracert/traceroute, ipconfig/ifconfig.
    • Wireshark (analisis paket).
  • Disk & Storage
    • chkdsk, Disk Management, SMART tools.
  • Security & Antivirus
    • Windows Defender, Malwarebytes, ClamAV.
  • Remote Support
    • TeamViewer, AnyDesk, Remote Desktop.

📊 Contoh Tools & Fungsinya

Tool Fungsi
ping Menguji konektivitas jaringan
Task Manager Memantau proses & penggunaan resource
chkdsk Memeriksa & memperbaiki error disk
Wireshark Menganalisis lalu lintas jaringan
TeamViewer Mendukung remote troubleshooting

🛠️ Praktik Kecil

Coba gunakan beberapa tools berikut:

  1. Buka Command Prompt, jalankan ping google.com.
  2. Buka Task Manager, lihat aplikasi yang menggunakan CPU paling tinggi.
  3. Jalankan chkdsk pada drive (read-only mode).

👉 Catat hasilnya dan apa yang bisa kamu simpulkan.

🎮 Mini Challenge

Buat skenario troubleshooting:

  • User mengeluh internet lambat → gunakan ping & tracert.
  • Komputer sering hang → cek Task Manager & Event Viewer.
  • File tidak bisa dibuka → gunakan chkdsk & cek SMART disk.

Tugas tambahan: Buat daftar 5 tools favoritmu untuk troubleshooting dan jelaskan kapan kamu menggunakannya.
