💻 Hardware Troubleshooting

Level: Beginner

Hardware Troubleshooting adalah proses mendiagnosis dan memperbaiki masalah pada perangkat fisik komputer. Bayangkan seperti montir mobil: ketika mesin tidak menyala, montir akan cek bensin, aki, kabel, lalu komponen mesin. Begitu juga IT Support memeriksa komponen hardware satu per satu.

📖 Jenis Masalah Hardware Umum

  • Power: komputer tidak menyala, PSU rusak, kabel longgar.
  • Storage: hard disk/SSD tidak terbaca, bad sector.
  • Memory (RAM): komputer sering restart, blue screen.
  • Display: monitor gelap, kabel VGA/HDMI rusak.
  • Peripheral: keyboard/mouse tidak berfungsi, printer error.
  • Overheating: kipas rusak, pasta thermal kering.

🧭 Metode Diagnosis

  1. Cek fisik: pastikan kabel terhubung, perangkat menyala.
  2. Swap test: ganti komponen dengan yang diketahui normal (misalnya RAM lain).
  3. POST/Beep code: dengarkan bunyi beep saat booting untuk indikasi error.
  4. Diagnostic tools: gunakan software bawaan (BIOS, vendor tools).
  5. Isolasi masalah: lepaskan komponen tambahan, jalankan hanya dengan minimal hardware.

📊 Contoh Gejala & Penyebab

Gejala Penyebab Umum Solusi Awal
Komputer tidak menyala Kabel power lepas, PSU rusak Cek kabel, coba PSU lain
Layar gelap Kabel display rusak, GPU bermasalah Coba kabel/monitor lain
Sering restart RAM error, overheating Cek RAM, bersihkan kipas

🛠️ Praktik Kecil

Coba lakukan simulasi:

  1. Lepaskan kabel monitor dari PC/laptop.
  2. Nyalakan perangkat, perhatikan gejala (layar gelap).
  3. Gunakan metode troubleshooting: cek kabel → pasang kembali → verifikasi layar menyala.

🎮 Mini Challenge

Pilih salah satu masalah berikut dan buat langkah troubleshooting:

  • Laptop cepat panas & mati sendiri.
  • Hard disk berbunyi aneh & tidak terbaca.
  • Printer tidak merespons perintah cetak.

Tugas tambahan: Buat checklist hardware troubleshooting standar untuk kantor kecil (power, kabel, RAM, storage, peripheral).
