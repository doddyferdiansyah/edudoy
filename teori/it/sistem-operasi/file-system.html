🗄️ File System Management

Level: Intermediate

File system adalah cara OS menyimpan dan mengatur data di media penyimpanan. Bayangkan harddisk atau SSD seperti sebuah perpustakaan besar, dan file system adalah sistem katalog yang memastikan setiap buku (file) punya tempat dan bisa ditemukan kembali.

📖 Penjelasan Singkat

  • Jenis File System: FAT32, NTFS (Windows), EXT4 (Linux), APFS (Mac).
  • Mounting: proses “menghubungkan” media penyimpanan agar bisa diakses OS.
  • Journaling: fitur untuk menjaga konsistensi data jika terjadi crash.
  • Metadata: informasi tentang file (ukuran, tanggal, permission).

Tanpa file system, data hanya berupa kumpulan bit yang tidak terstruktur.

🛠️ Praktik Kecil

Coba cek tipe file system di komputer kamu:

# Windows (Command Prompt)
fsutil fsinfo volumeinfo C:

# Linux/Mac (Terminal)
df -T

👉 Dari sini kamu bisa tahu apakah drive kamu menggunakan NTFS, FAT32, EXT4, atau lainnya.

🎮 Mini Challenge

Bayangkan kamu punya flashdisk yang ingin dipakai di Windows, Linux, dan Mac.

Tugas: Tentukan file system apa yang sebaiknya digunakan agar kompatibel di semua OS, dan jelaskan kelebihan serta kekurangannya.
