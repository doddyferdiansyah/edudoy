Level: Advanced

Virtualisasi memungkinkan satu komputer fisik menjalankan banyak sistem operasi sekaligus. Bayangkan sebuah rumah besar yang dibagi menjadi beberapa apartemen: setiap penghuni (OS) merasa punya rumah sendiri, padahal mereka berbagi bangunan yang sama (hardware).

ğŸ“– Penjelasan Singkat

  • Hypervisor Type 1: berjalan langsung di atas hardware (contoh: VMware ESXi, Microsoft Hyper-V bare metal).
  • Hypervisor Type 2: berjalan di atas OS host (contoh: VirtualBox, VMware Workstation).
  • Container: virtualisasi di level aplikasi, lebih ringan daripada VM (contoh: Docker, LXC).
  • Isolasi: setiap VM/container terpisah sehingga lebih aman dan fleksibel.

Virtualisasi membuat resource lebih efisien, memudahkan testing, dan mendukung cloud computing.
