Ada dua cara utama berinteraksi dengan sistem operasi:

  • GUI (Graphical User Interface): antarmuka grafis dengan ikon, jendela, dan menu. Contoh: Windows Explorer.
  • CLI (Command Line Interface): antarmuka berbasis teks. Contoh: Terminal di Linux, Command Prompt di Windows.

Keduanya sama-sama penting: GUI mudah dipahami, CLI lebih cepat dan fleksibel bagi yang terbiasa.
