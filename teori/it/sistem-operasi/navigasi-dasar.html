🖱️ Navigasi Dasar Sistem Operasi

Level: Beginner

Bayangkan komputer seperti sebuah kota besar. Agar tidak tersesat, kita butuh peta dan cara berpindah dari satu tempat ke tempat lain. Navigasi dasar OS adalah keterampilan untuk menjelajahi “kota” komputer, baik lewat jalan utama (GUI) maupun jalan pintas (CLI).

📖 Penjelasan Singkat

Ada dua cara utama berinteraksi dengan sistem operasi:

  • GUI (Graphical User Interface): antarmuka grafis dengan ikon, jendela, dan menu. Contoh: Windows Explorer.
  • CLI (Command Line Interface): antarmuka berbasis teks. Contoh: Terminal di Linux, Command Prompt di Windows.

Keduanya sama-sama penting: GUI mudah dipahami, CLI lebih cepat dan fleksibel bagi yang terbiasa.

🛠️ Praktik Kecil

Coba lakukan navigasi sederhana:

# Windows (Command Prompt)
cd C:\
dir

# Linux/Mac (Terminal)
cd /
ls

👉 Perintah ini akan memindahkan kamu ke direktori utama (root) dan menampilkan daftar folder di dalamnya.

🎮 Mini Challenge

Bayangkan kamu sedang menjelajahi sebuah kota:

  • GUI = naik mobil lewat jalan raya (nyaman, tapi kadang macet).
  • CLI = naik sepeda motor lewat gang kecil (lebih cepat, tapi butuh tahu jalannya).

Tugas: Coba jelaskan kapan kamu lebih memilih GUI, dan kapan lebih memilih CLI dalam kehidupan sehari-hari.
