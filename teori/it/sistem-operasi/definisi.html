OS bertugas menjadi jembatan antara pengguna dengan perangkat keras. Ia mengatur bagaimana aplikasi berjalan, bagaimana file disimpan, dan bagaimana perangkat keras seperti CPU, RAM, dan harddisk bekerja sama.

  • Resource Manager: mengatur CPU, memori, dan perangkat I/O.
  • User Interface: menyediakan antarmuka (CLI atau GUI) agar kita bisa berinteraksi.
  • Security & Access Control: menjaga agar hanya pengguna yang berhak bisa mengakses data.
