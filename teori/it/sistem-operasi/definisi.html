🖥️ Definisi Sistem Operasi

Level: Beginner

Sistem Operasi (Operating System/OS) adalah “otak” dari komputer. Tanpa OS, komputer hanyalah kumpulan perangkat keras yang tidak bisa berkomunikasi dengan kita. Bayangkan OS seperti dirigen orkestra yang mengatur semua instrumen (hardware & software) agar menghasilkan musik yang harmonis.

📖 Penjelasan Singkat

OS bertugas menjadi jembatan antara pengguna dengan perangkat keras. Ia mengatur bagaimana aplikasi berjalan, bagaimana file disimpan, dan bagaimana perangkat keras seperti CPU, RAM, dan harddisk bekerja sama.

  • Resource Manager: mengatur CPU, memori, dan perangkat I/O.
  • User Interface: menyediakan antarmuka (CLI atau GUI) agar kita bisa berinteraksi.
  • Security & Access Control: menjaga agar hanya pengguna yang berhak bisa mengakses data.

🛠️ Praktik Kecil

Coba buka Command Prompt (Windows) atau Terminal (Linux/Mac), lalu ketik perintah berikut:

whoami

👉 Perintah ini akan menampilkan nama user yang sedang login. Dari sini siswa bisa memahami bahwa OS mengenali setiap pengguna.

🎮 Mini Challenge

Bayangkan komputer kamu adalah sebuah restoran:

  • CPU = koki
  • RAM = meja dapur
  • Harddisk = gudang bahan makanan
  • OS = manajer restoran

Tugas: Jelaskan dengan kata-kata sendiri bagaimana OS (manajer) mengatur koki, meja, dan gudang agar restoran berjalan lancar.
