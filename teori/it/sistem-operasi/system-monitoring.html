Level: Intermediate

Sistem Operasi tidak hanya menjalankan aplikasi, tapi juga mencatat dan memantau semua aktivitas. Bayangkan OS seperti kamera CCTV dan papan kontrol di sebuah gedung: CCTV (logging) merekam semua kejadian, sedangkan papan kontrol (monitoring) menunjukkan kondisi gedung secara real-time.

ğŸ“– Penjelasan Singkat

  • Monitoring: memantau penggunaan CPU, RAM, disk, dan jaringan.
  • Logging: mencatat semua event penting (error, login, update sistem).
  • Tools:
    • Windows: Task Manager, Event Viewer
    • Linux: top/htop, journalctl, syslog

Monitoring membantu mendeteksi masalah secara cepat, logging membantu menelusuri penyebab masalah.
