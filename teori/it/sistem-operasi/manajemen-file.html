Level: Beginner

File adalah wadah informasi di komputer. Sistem Operasi bertugas mengatur bagaimana file dibuat, disimpan, diakses, dan dihapus. Bayangkan file seperti lembaran kertas di dalam sebuah lemari arsip. OS adalah petugas arsip yang memastikan semua kertas tersusun rapi dan mudah ditemukan.

ğŸ“– Penjelasan Singkat

Manajemen file mencakup:

  • Penyimpanan: bagaimana file disimpan di harddisk/SSD.
  • Penamaan: aturan nama file dan ekstensi.
  • Struktur Direktori: folder dan subfolder untuk mengorganisasi file.
  • Akses: siapa yang boleh membaca, menulis, atau menghapus file.
