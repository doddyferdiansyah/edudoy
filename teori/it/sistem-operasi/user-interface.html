🖥️ User Interface (CLI & GUI)

Level: Beginner

User Interface (UI) adalah cara kita berkomunikasi dengan komputer. Ada dua jalur utama: GUI (Graphical User Interface) yang penuh ikon dan jendela, serta CLI (Command Line Interface) yang berbasis teks. Bayangkan GUI seperti jalan tol yang nyaman, sedangkan CLI seperti jalan tikus yang lebih cepat jika tahu jalannya.

📖 Penjelasan Singkat

  • GUI: mudah dipahami, cocok untuk pemula. Contoh: klik ikon folder untuk membuka file.
  • CLI: lebih cepat, fleksibel, dan powerful. Contoh: mengetik perintah untuk membuka file.

Keduanya saling melengkapi. Profesional IT biasanya menguasai keduanya agar lebih efisien.

🛠️ Praktik Kecil

Coba buka aplikasi Notepad dengan dua cara:

# GUI
Klik Start Menu → ketik "Notepad" → Enter

# CLI (Windows Command Prompt)
notepad

# CLI (Linux Terminal)
nano catatan.txt

👉 Dengan ini, siswa bisa merasakan perbedaan antara GUI dan CLI.

🎮 Mini Challenge

Bayangkan kamu sedang bepergian:

  • GUI = naik mobil lewat jalan raya (nyaman, tapi kadang macet).
  • CLI = naik motor lewat gang kecil (lebih cepat, tapi butuh tahu jalannya).

Tugas: Tuliskan satu contoh aktivitas di komputer yang lebih enak dilakukan dengan GUI, dan satu contoh yang lebih cepat dengan CLI.
