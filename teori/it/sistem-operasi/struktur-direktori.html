📂 Struktur Direktori

Level: Beginner

Struktur direktori adalah peta kota dari komputer. Setiap file dan folder punya alamat yang jelas, sehingga kita bisa menemukan dan mengatur data dengan rapi. Bayangkan folder sebagai jalan dan file sebagai rumah di dalam kota digital.

📖 Penjelasan Singkat

Struktur direktori berbeda di tiap sistem operasi:

  • Windows: biasanya dimulai dari C:\ (contoh: C:\Users\Doddy\Documents).
  • Linux/Mac: dimulai dari / (root), lalu bercabang ke /home, /etc, /var, dll.

Dengan memahami struktur ini, kita bisa lebih mudah mengelola file dan menghindari kekacauan.

🛠️ Praktik Kecil

Coba tampilkan struktur direktori dasar:

# Windows (Command Prompt)
tree C:\ /F

# Linux/Mac (Terminal)
ls /

👉 Perintah ini akan menampilkan isi direktori utama. Dari sini, siswa bisa melihat bagaimana folder disusun secara hierarki.

🎮 Mini Challenge

Bayangkan kamu sedang membuat folder untuk proyek sekolah/kuliah. Kamu ingin menyimpan dokumen, gambar, dan presentasi dengan rapi.

Tugas: Buatlah struktur folder sederhana seperti ini:

Proyek/
 ├── Dokumen/
 ├── Gambar/
 └── Presentasi/

👉 Tuliskan bagaimana kamu akan menambahkan file ke dalam masing-masing folder.
