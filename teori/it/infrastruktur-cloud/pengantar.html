Level: Beginner

Infrastruktur IT adalah fondasi teknologi yang mendukung aplikasi dan layanan digital. Cloud Computing adalah model penyediaan infrastruktur, platform, dan software melalui internet. Bayangkan cloud seperti listrik: kita tidak perlu membangun pembangkit sendiri, cukup berlangganan dan pakai sesuai kebutuhan.

ğŸ“– Konsep Dasar

  • Infrastruktur Tradisional (On-Premise): server fisik, storage, dan jaringan dikelola sendiri.
  • Cloud Computing: sumber daya (server, storage, aplikasi) disediakan oleh penyedia layanan cloud.
  • Karakteristik Cloud:
    • On-demand self-service (pakai sesuai kebutuhan).
    • Scalability (mudah diperbesar/diperkecil).
    • Pay-as-you-go (bayar sesuai pemakaian).
    • Akses melalui internet dari mana saja.
