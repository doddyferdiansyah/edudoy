Level: Beginner

Cloud Providers adalah perusahaan penyedia layanan cloud yang menawarkan infrastruktur, platform, dan software melalui internet. Tiga pemain utama adalah AWS, Microsoft Azure, dan Google Cloud Platform (GCP). Bayangkan mereka seperti penyedia listrik di kota: sama-sama memberi energi, tapi dengan paket dan layanan berbeda.

ğŸ“– Tiga Cloud Provider Utama

  • AWS (Amazon Web Services)
    • Pionir cloud publik, layanan paling luas & matang.
    • Kuat di infrastruktur (IaaS) & layanan enterprise.
    • Contoh layanan: EC2 (VM), S3 (storage), RDS (database).
  • Microsoft Azure
    • Integrasi kuat dengan ekosistem Microsoft (Windows Server, Active Directory, Office 365).
    • Populer di perusahaan yang sudah pakai produk Microsoft.
    • Contoh layanan: Azure VM, Blob Storage, Azure SQL Database.
  • Google Cloud Platform (GCP)
    • Kuat di data analytics, machine learning, dan container (Kubernetes lahir dari Google).
    • Populer di startup & perusahaan data-driven.
    • Contoh layanan: Compute Engine, Cloud Storage, BigQuery.
