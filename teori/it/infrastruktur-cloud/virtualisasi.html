Level: Beginner

Virtualisasi adalah teknologi yang memungkinkan satu komputer fisik menjalankan beberapa sistem operasi secara bersamaan melalui mesin virtual (VM). Bayangkan sebuah rumah besar (server fisik) yang dibagi menjadi beberapa apartemen (VM), masing-masing bisa dihuni dengan aturan dan dekorasi berbeda.

ğŸ“– Konsep Dasar Virtualisasi

  • Hypervisor: perangkat lunak yang mengatur VM (contoh: VMware, VirtualBox, KVM, Hyper-V).
  • Mesin Virtual (VM): sistem operasi lengkap yang berjalan di atas hypervisor.
  • Manfaat:
    • Efisiensi penggunaan hardware.
    • Isolasi antar sistem (jika satu crash, yang lain tetap jalan).
    • Mudah untuk testing & development.

ğŸ”„ VM vs Container

  • VM: butuh OS lengkap, lebih berat, isolasi kuat.
  • Container: berbagi kernel host, lebih ringan, startup cepat.

ğŸ‘‰ VM cocok untuk workload besar & legacy apps, sedangkan container cocok untuk microservices & aplikasi modern.

Coba buat VM menggunakan VirtualBox:

  1. Install VirtualBox di laptop/PC.

  3. Atur resource (CPU, RAM, storage).
  4. Jalankan VM dan coba akses terminal Linux di dalamnya.
