Level: Beginner

Fungsi adalah blok kode yang dirancang untuk melakukan tugas tertentu. Dengan fungsi, kita bisa menghindari penulisan kode berulang. Bayangkan seperti alat dapur: blender hanya perlu ditekan tombolnya untuk bekerja, kita tidak perlu tahu detail mekanismenya setiap kali.

ğŸ“– Penjelasan Singkat

  • Definisi: dibuat dengan kata kunci def di Python.
  • Parameter: input yang diberikan ke fungsi.
  • Return: nilai yang dikembalikan oleh fungsi.

Fungsi membuat kode lebih terstruktur, mudah dibaca, dan bisa digunakan kembali.

Coba jalankan kode Python berikut:

# Membuat fungsi

    return "Halo, " + nama + "!"

# Memanggil fungsi
print(sapa("Doddy"))
print(sapa("Budi"))
