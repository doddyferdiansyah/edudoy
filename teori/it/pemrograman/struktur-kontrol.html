🔀 Struktur Kontrol

Level: Beginner

Struktur kontrol adalah cara kita mengatur alur eksekusi program. Dengan struktur kontrol, kita bisa membuat komputer mengambil keputusan atau mengulang instruksi. Bayangkan seperti rambu lalu lintas: kadang harus belok, kadang berhenti, kadang jalan terus.

📖 Jenis Struktur Kontrol

  • Kondisional (if, elif, else): memilih jalur berdasarkan kondisi tertentu.
  • Perulangan (for, while): mengulang instruksi beberapa kali.
  • Kontrol alur tambahan: break, continue, pass.

Struktur kontrol membuat program lebih dinamis dan mampu menangani berbagai situasi.

🛠️ Praktik Kecil

Coba jalankan kode Python berikut:

# Kondisional
nilai = 75
if nilai >= 80:
    print("Lulus dengan baik")
elif nilai >= 60:
    print("Lulus")
else:
    print("Tidak lulus")

# Perulangan
for i in range(3):
    print("Belajar Python ke-", i+1)

👉 Kondisional akan memilih output sesuai nilai, sedangkan perulangan akan mengulang teks 3 kali.

🎮 Mini Challenge

Buat program yang meminta input umur dari pengguna, lalu tampilkan:

  • "Anak-anak" jika umur < 12
  • "Remaja" jika umur 12–17
  • "Dewasa" jika umur 18–59
  • "Lansia" jika umur ≥ 60

Tugas: Gunakan if-elif-else untuk menentukan kategori umur.
