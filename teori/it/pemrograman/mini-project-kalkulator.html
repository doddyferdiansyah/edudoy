Level: Beginner

Setelah mempelajari variabel, operator, struktur kontrol, fungsi, dan input-output, saatnya kita gabungkan semua konsep itu dalam sebuah kalkulator sederhana.

ğŸ“– Deskripsi Proyek

Kalkulator ini akan meminta pengguna memilih operasi (tambah, kurang, kali, bagi), lalu memasukkan dua angka, dan menampilkan hasilnya.

ğŸ› ï¸ Implementasi

def tambah(a, b):
    return a + b

def kurang(a, b):
    return a - b

def kali(a, b):
    return a * b

def bagi(a, b):
    if b != 0:
        return a / b
    else:
        return "Error: Pembagian dengan nol!"

print("=== Kalkulator Sederhana ===")
print("1. Tambah")
print("2. Kurang")
print("3. Kali")
print("4. Bagi")

pilihan = input("Pilih operasi (1/2/3/4): ")

x = float(input("Masukkan angka pertama: "))
y = float(input("Masukkan angka kedua: "))

if pilihan == "1":
    print("Hasil:", tambah(x, y))
elif pilihan == "2":
    print("Hasil:", kurang(x, y))
elif pilihan == "3":
    print("Hasil:", kali(x, y))
elif pilihan == "4":
    print("Hasil:", bagi(x, y))
else:
    print("Pilihan tidak valid")
