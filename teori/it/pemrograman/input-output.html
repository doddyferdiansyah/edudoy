⌨️ Input & Output

Level: Beginner

Input adalah cara program menerima data dari pengguna, sedangkan Output adalah cara program menampilkan hasil ke layar. Bayangkan seperti percakapan: kita bertanya (input), lalu menjawab (output).

📖 Penjelasan Singkat

  • input() → menerima data dari pengguna (selalu berupa string).
  • print() → menampilkan data ke layar.
  • Data dari input() bisa diubah ke tipe lain, misalnya int() atau float().

Dengan input-output, program menjadi interaktif dan bisa berkomunikasi dengan pengguna.

🛠️ Praktik Kecil

Coba jalankan kode Python berikut:

nama = input("Siapa namamu? ")
print("Halo,", nama)

umur = int(input("Berapa umurmu? "))
print("Umurmu tahun depan adalah", umur + 1)

👉 Program akan menunggu input dari pengguna, lalu menampilkan hasil sesuai data yang dimasukkan.

🎮 Mini Challenge

Buat program yang meminta pengguna memasukkan dua angka, lalu tampilkan hasil penjumlahannya.

Tugas tambahan: Ubah program agar juga menampilkan hasil perkalian dari dua angka tersebut.
