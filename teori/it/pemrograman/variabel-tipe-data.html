📦 Variabel & Tipe Data

Level: Beginner

Variabel adalah wadah untuk menyimpan data, sedangkan tipe data adalah jenis isi dari wadah tersebut. Bayangkan seperti kotak penyimpanan: ada kotak khusus untuk angka, teks, atau benar/salah. Dengan variabel, kita bisa memberi nama pada data agar mudah digunakan kembali.

📖 Penjelasan Singkat

  • Integer: bilangan bulat (contoh: 10, -5).
  • Float: bilangan desimal (contoh: 3.14, -0.5).
  • String: teks (contoh: "Halo Dunia").
  • Boolean: logika benar/salah (True atau False).

Variabel membantu kita menyimpan nilai agar bisa dipanggil kapan saja tanpa menulis ulang.

🛠️ Praktik Kecil

Coba jalankan kode Python berikut:

# Contoh variabel
nama = "Doddy"
umur = 25
tinggi = 170.5
is_student = True

print(nama, umur, tinggi, is_student)

👉 Output akan menampilkan isi variabel sesuai tipe datanya.

🎮 Mini Challenge

Kamu ingin menyimpan informasi tentang sebuah buku: judul, jumlah halaman, harga, dan status apakah masih tersedia.

Tugas: Buat 4 variabel dengan tipe data yang sesuai untuk menyimpan informasi tersebut, lalu tampilkan dengan print().
