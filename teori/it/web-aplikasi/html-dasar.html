📄 HTML Dasar

Level: Beginner

HTML (HyperText Markup Language) adalah bahasa standar untuk membuat struktur halaman web. Bayangkan HTML seperti kerangka rumah: ia menentukan ruangan, pintu, dan jendela, sebelum diberi cat (CSS) dan perabot (JavaScript).

📖 Struktur Dasar HTML

<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
  </body>
</html>

🔑 Tag Penting

  • <h1> ... </h1> → judul utama (heading).
  • <p> ... </p> → paragraf.
  • <a href="..."> ... </a> → hyperlink.
  • <img src="..." alt="..."> → gambar.
  • <ul><li>...</li></ul> → daftar bullet.
  • <ol><li>...</li></ol> → daftar bernomor.

🛠️ Praktik Kecil

Buat file index.html sederhana:

<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8">
    <title>Profil Saya</title>
  </head>
  <body>
    <h1>Halo, saya Doddy</h1>
    <p>Saya sedang belajar membuat website dengan HTML.</p>
    <img src="foto.jpg" alt="Foto Profil">
    <a href="https://fixly.com">Kunjungi Fixly</a>
  </body>
</html>

👉 Simpan file ini lalu buka di browser untuk melihat hasilnya.

🎮 Mini Challenge

Buat halaman HTML sederhana dengan ketentuan:

  • Judul halaman: "Belajar HTML".
  • Heading utama berisi nama kamu.
  • Paragraf yang menjelaskan hobi kamu.
  • Tambahkan gambar (boleh placeholder).
  • Buat daftar 3 makanan favoritmu.
  • Tambahkan link ke website favoritmu.

Tugas tambahan: Gunakan kombinasi <ol> dan <ul> untuk membuat daftar bertingkat.
