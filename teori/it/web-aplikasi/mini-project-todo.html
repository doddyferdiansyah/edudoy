✅ Mini Project: To-Do List

Level: Intermediate

Pada mini project ini, kita akan membuat aplikasi To-Do List sederhana. Aplikasi ini memungkinkan pengguna menambah, menandai selesai, dan menghapus tugas. Bayangkan seperti catatan harian digital untuk mengatur aktivitas.

📖 Fitur Utama

  • Menambahkan tugas baru.
  • Menandai tugas selesai (strike-through).
  • Menghapus tugas dari daftar.

🛠️ Praktik Kecil

To-Do List

    👉 Coba tambahkan tugas, klik untuk menandai selesai, dan gunakan tombol hapus untuk menghapus tugas.

    🎮 Mini Challenge

    Kembangkan aplikasi To-Do List dengan fitur tambahan:

    • Tambahkan tombol "Hapus Semua" untuk menghapus semua tugas sekaligus.
    • Simpan daftar tugas di localStorage agar tidak hilang saat halaman direfresh.
    • Tambahkan filter: tampilkan semua tugas, hanya yang selesai, atau hanya yang belum selesai.

    Tugas tambahan: Tambahkan animasi sederhana saat menambah atau menghapus tugas.
