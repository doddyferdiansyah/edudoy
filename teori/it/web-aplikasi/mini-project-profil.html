Kembangkan halaman profil dengan fitur tambahan:

  • Tambahkan bagian "Tentang Saya" dengan paragraf lebih panjang.
  • Tambahkan tabel sederhana berisi daftar pengalaman kerja atau pendidikan.
  • Tambahkan tombol "Hubungi Saya" yang mengarah ke email.
  • Gunakan CSS untuk membuat tampilan lebih menarik (warna, font, hover effect).

Tugas tambahan: Buat halaman profil menjadi responsif dengan CSS media query agar tetap rapi di layar HP.
