👤 Mini Project: Halaman Profil

Level: Beginner

Pada mini project ini, kita akan membuat halaman profil sederhana menggunakan HTML dan CSS. Bayangkan ini sebagai kartu nama digital yang menampilkan identitas, hobi, dan link sosial media.

📖 Struktur Halaman

Halaman profil akan berisi:

  • Foto profil
  • Nama & deskripsi singkat
  • Daftar hobi
  • Link ke media sosial

🛠️ Praktik Kecil

Contoh kode HTML + CSS untuk halaman profil:

<div class="container">
  <img src="foto.jpg" alt="Foto Profil" class="profile-pic">
  <h1>Doddy</h1>
  <p>IT Support Specialist & Content Creator</p>

  <h3>Hobi</h3>
  <ul>
    <li>Coding</li>
    <li>Membaca</li>
    <li>Gaming</li>
  </ul>

  <h3>Temukan Saya</h3>
  <ul>
    <li><a href="https://github.com">GitHub</a></li>
    <li><a href="https://linkedin.com">LinkedIn</a></li>
  </ul>
</div>

👉 Simpan file ini sebagai profil.html dan buka di browser.

🎮 Mini Challenge

Kembangkan halaman profil dengan fitur tambahan:

  • Tambahkan bagian "Tentang Saya" dengan paragraf lebih panjang.
  • Tambahkan tabel sederhana berisi daftar pengalaman kerja atau pendidikan.
  • Tambahkan tombol "Hubungi Saya" yang mengarah ke email.
  • Gunakan CSS untuk membuat tampilan lebih menarik (warna, font, hover effect).

Tugas tambahan: Buat halaman profil menjadi responsif dengan CSS media query agar tetap rapi di layar HP.
