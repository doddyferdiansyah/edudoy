CSS (Cascading Style Sheets) digunakan untuk memberikan gaya pada halaman HTML. Bayangkan CSS seperti cat dan dekorasi rumah: HTML adalah kerangka, CSS membuatnya indah dan nyaman dilihat.

  • Inline CSS: langsung di dalam tag HTML. <p style="color:red;">Teks merah</p>
  • Internal CSS: ditulis di dalam tag <style> di file HTML. 
    <style>
p { color: blue; }
</style>
  • External CSS: ditulis di file terpisah .css lalu dihubungkan dengan <link>. 
    <link rel="stylesheet" href="style.css">
