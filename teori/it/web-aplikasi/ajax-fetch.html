🌐 AJAX & Fetch API

Level: Intermediate

AJAX (Asynchronous JavaScript and XML) memungkinkan web mengambil/mengirim data ke server tanpa reload halaman. Fetch API adalah cara modern untuk melakukan AJAX dengan sintaks lebih sederhana. Bayangkan seperti kurir instan: data bisa dikirim/diterima tanpa harus menutup pintu (reload halaman).

📖 Konsep Dasar

  • AJAX → teknik lama menggunakan XMLHttpRequest.
  • Fetch API → cara baru, berbasis Promise, lebih mudah dibaca.
  • JSON → format data paling umum untuk pertukaran client-server.

🛠️ Praktik Kecil

    👉 Klik tombol untuk mengambil data user dari API publik tanpa reload halaman.

    🎮 Mini Challenge

    Buat aplikasi sederhana dengan Fetch API:

    • Ambil daftar posts dari https://jsonplaceholder.typicode.com/posts.
    • Tampilkan judul dan isi postingan di halaman.
    • Tambahkan tombol "Muat Lagi" untuk menampilkan 5 postingan berikutnya.

    Tugas tambahan: Buat form sederhana untuk menambahkan postingan baru (gunakan fetch POST) dan tampilkan hasilnya di halaman.
