📡 Model OSI & TCP/IP

Level: Intermediate

Model OSI dan TCP/IP adalah kerangka kerja yang menjelaskan bagaimana data bergerak di jaringan. Bayangkan seperti kantor pos: ada proses menulis surat, memasukkan ke amplop, memberi alamat, mengirim, hingga diterima. Setiap tahap punya peran sendiri agar pesan sampai dengan benar.

📖 Penjelasan Singkat

Model OSI (7 Layer)

  1. Application: layanan untuk user (HTTP, FTP, SMTP).
  2. Presentation: format data, enkripsi, kompresi.
  3. Session: mengatur sesi komunikasi.
  4. Transport: memastikan data sampai utuh (TCP/UDP).
  5. Network: pengalamatan & routing (IP).
  6. Data Link: pengiriman frame antar perangkat (Ethernet, MAC).
  7. Physical: media fisik (kabel, sinyal, radio).

Model TCP/IP (4 Layer)

  1. Application: HTTP, FTP, DNS.
  2. Transport: TCP, UDP.
  3. Internet: IP, ICMP.
  4. Network Access: Ethernet, Wi-Fi.

Model OSI lebih detail (7 layer), sedangkan TCP/IP lebih praktis (4 layer) dan digunakan di dunia nyata.

🛠️ Praktik Kecil

Coba lakukan ping untuk melihat bagaimana protokol bekerja:

# Windows / Linux / Mac
ping google.com

👉 Saat kamu melakukan ping, data melewati beberapa layer: Application (ICMP request), Transport, Internet (IP), hingga Network Access.
