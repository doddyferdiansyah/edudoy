📋 Struktur Tabel & Tipe Data

Level: Beginner

Tabel adalah struktur utama dalam database relasional. Tabel terdiri dari kolom (field) dan baris (record). Bayangkan seperti lembar Excel: kolom = kategori data, baris = data aktual.

📖 Komponen Tabel

  • Kolom (Field): mendefinisikan jenis data (misalnya nama, umur, alamat).
  • Baris (Record): menyimpan data aktual sesuai kolom.
  • Primary Key: kolom unik untuk identifikasi tiap baris.
  • Tipe Data: menentukan jenis data yang bisa disimpan (INT, VARCHAR, DATE, dll).

🔑 Contoh Tipe Data Umum

  • INT → bilangan bulat
  • VARCHAR(n) → teks dengan panjang maksimum n
  • DATE → tanggal (YYYY-MM-DD)
  • BOOLEAN → true/false

🛠️ Praktik Kecil

Buat tabel mahasiswa dengan struktur berikut:

CREATE TABLE mahasiswa (
    id INT PRIMARY KEY AUTO_INCREMENT,
    nama VARCHAR(50) NOT NULL,
    umur INT,
    jurusan VARCHAR(30),
    tanggal_daftar DATE
);

👉 Tabel di atas memiliki kolom id sebagai primary key, serta kolom teks, angka, dan tanggal.

🎮 Mini Challenge

Buat tabel pegawai dengan kolom:

  • id (primary key, auto increment)
  • nama (varchar 50, wajib diisi)
  • posisi (varchar 30)
  • gaji (int)
  • tanggal_masuk (date)

Tugas tambahan: Tambahkan constraint agar kolom gaji tidak boleh bernilai negatif.
