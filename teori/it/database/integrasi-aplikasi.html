Level: Advanced

Database jarang berdiri sendiri â€” biasanya digunakan oleh aplikasi. Integrasi database dengan aplikasi memungkinkan data disimpan, diambil, dan dimanipulasi melalui kode program. Bayangkan seperti jembatan antara aplikasi frontend dan gudang data di backend.

ğŸ“– Konsep Integrasi

  • Driver/Connector: library yang memungkinkan aplikasi berkomunikasi dengan database (contoh: MySQL Connector, psycopg2 untuk PostgreSQL).
  • Query Execution: aplikasi mengirim perintah SQL ke database.
  • ORM (Object Relational Mapping): cara modern untuk mengelola database dengan objek (contoh: Sequelize, SQLAlchemy, Hibernate).

ğŸ› ï¸ Praktik Kecil

Contoh integrasi database MySQL dengan Python menggunakan mysql-connector:

import mysql.connector

# Koneksi ke database
db = mysql.connector.connect(
    host="localhost",
    user="root",
    password="passwordku",
    database="inventaris_db"
)

cursor = db.cursor()

# Membaca data
cursor.execute("SELECT nama_barang, stok FROM barang")
for (nama, stok) in cursor.fetchall():
    print(f"{nama} - Stok: {stok}")

# Menambahkan data
cursor.execute(
    "INSERT INTO barang (nama_barang, stok, kategori_id) VALUES (%s, %s, %s)",
    ("Keyboard", 15, 1)
)
db.commit()

print("Data berhasil ditambahkan!")

ğŸ® Mini Challenge

Buat aplikasi sederhana (boleh Python, Node.js, atau PHP) yang terhubung ke database inventaris_db:

  • Tampilkan daftar barang beserta stoknya.
  • Tambahkan fitur untuk menambah barang baru.
  • Tambahkan fitur untuk memperbarui stok barang tertentu.

Tugas tambahan: Gunakan ORM (misalnya SQLAlchemy di Python atau Sequelize di Node.js) agar kode lebih rapi dan mudah dipelihara.
