💻 SQL Dasar

Level: Beginner

SQL (Structured Query Language) adalah bahasa standar untuk mengelola database relasional. Dengan SQL, kita bisa menambahkan, membaca, memperbarui, dan menghapus data. Bayangkan seperti remote control untuk database: kita bisa memberi perintah sesuai kebutuhan.

📖 Perintah SQL Dasar

  • SELECT → mengambil data dari tabel.
  • INSERT → menambahkan data baru.
  • UPDATE → memperbarui data yang sudah ada.
  • DELETE → menghapus data.

🛠️ Praktik Kecil

Contoh penggunaan SQL dasar pada tabel mahasiswa:

-- Menambahkan data
INSERT INTO mahasiswa (nama, umur, jurusan, tanggal_daftar)
VALUES ('Budi', 20, 'Informatika', '2025-01-01');

-- Membaca data
SELECT * FROM mahasiswa;

-- Memperbarui data
UPDATE mahasiswa
SET umur = 21
WHERE nama = 'Budi';

-- Menghapus data
DELETE FROM mahasiswa
WHERE nama = 'Budi';

👉 Perintah di atas menambahkan data mahasiswa, menampilkan semua data, memperbarui umur, lalu menghapus data.

🎮 Mini Challenge

Buat tabel pegawai (jika belum ada), lalu lakukan hal berikut:

  • Tambahkan 3 data pegawai baru.
  • Tampilkan semua data pegawai.
  • Perbarui posisi salah satu pegawai.
  • Hapus 1 pegawai dari tabel.

Tugas tambahan: Gunakan WHERE untuk memastikan hanya data tertentu yang diperbarui atau dihapus.
