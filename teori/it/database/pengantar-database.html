📘 Pengantar Database

Level: Beginner

Database adalah sistem yang digunakan untuk menyimpan, mengelola, dan mengambil data secara terstruktur. Bayangkan seperti lemari arsip digital: setiap laci (tabel) berisi dokumen (baris) dengan label tertentu (kolom). Dengan database, data dapat diakses dengan cepat, aman, dan efisien.

📖 Mengapa Database Penting?

  • Organisasi Data: menyimpan data dalam bentuk tabel yang rapi.
  • Konsistensi: menjaga agar data tetap akurat dan tidak duplikat.
  • Keamanan: mengatur siapa yang boleh mengakses data.
  • Efisiensi: memudahkan pencarian dan manipulasi data.

Database digunakan di hampir semua aplikasi modern: e-commerce, media sosial, perbankan, hingga sistem akademik.

🛠️ Praktik Kecil

Coba buat database sederhana di MySQL atau SQLite:

-- Membuat database baru
CREATE DATABASE sekolah;

-- Menggunakan database
USE sekolah;

-- Membuat tabel siswa
CREATE TABLE siswa (
    id INT PRIMARY KEY AUTO_INCREMENT,
    nama VARCHAR(50),
    umur INT,
    kelas VARCHAR(10)
);

👉 Di atas kita membuat database sekolah dengan tabel siswa yang punya kolom id, nama, umur, dan kelas.

🎮 Mini Challenge

Buat database bernama perpustakaan dengan tabel buku yang memiliki kolom:

  • id (primary key, auto increment)
  • judul (teks)
  • penulis (teks)
  • tahun_terbit (angka)

Tugas tambahan: Tambahkan 3 data buku menggunakan perintah INSERT.
