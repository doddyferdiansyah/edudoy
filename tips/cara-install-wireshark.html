Level: Beginner

Wireshark adalah tool open-source untuk menganalisis paket jaringan. Di bawah ini panduan instalasi untuk Windows, Linux, dan macOS, plus verifikasi dan tips penggunaan awal.

ğŸªŸ Install di Windows 10/11

  1. Kunjungi situs resmi Wireshark dan unduh installer terbaru (64-bit).
  2. Jalankan installer, pilih Next hingga bagian komponen.
  3. Pastikan opsi Npcap dicentang (dibutuhkan untuk capture paket di Windows).
  4. Lanjutkan instalasi Npcap:
    • Centang Support raw 802.11 traffic (and monitor mode) for wireless adapters jika diperlukan.
    • Centang Install Npcap in WinPcap API-compatible Mode jika butuh kompatibilitas lama.
  5. Selesai instalasi, buka Wireshark dari Start Menu.
Catatan: Beberapa adapter Wi-Fi di Windows tidak mendukung monitor mode; capture terbaik biasanya via Ethernet atau dengan adapter Wi-Fi tertentu yang kompatibel.

ğŸ§ Install di Linux (Ubuntu/Debian)

  1. Update repo: 
    sudo apt update
  2. Install Wireshark: 
    sudo apt install wireshark
  3. Izinkan non-root melakukan capture: 
    sudo dpkg-reconfigure wireshark-common
sudo usermod -aG wireshark $USER
# logout/login atau reboot agar group berlaku

ğŸ§ Install di Linux (Fedora/RHEL/CentOS)

  1. Install paket: 
    sudo dnf install wireshark wireshark-cli
  2. Tambahkan user ke group: 
    sudo usermod -aG wireshark $USER
  3. Logout/login ulang sebelum capture.

ğŸš€ Mulai Capture Pertama

  1. Pilih interface aktif (misal: Ethernet atau Wi-Fi).
  2. Klik Start, lalu di browser akses example.com.
  3. Gunakan filter tampilan:
    • HTTP: http
    • DNS: dns
    • ICMP (ping): icmp
    • IP ke host tertentu: ip.addr == 8.8.8.8
  4. Klik paket dan lihat detail di panel tengah (header + payload).

  • Interface tidak muncul: di Windows, reinstall Npcap; di Linux, cek group wireshark dan izin.
  • Tidak ada paket: pastikan interface benar, atau matikan VPN yang mengalihkan traffic.
  • Izin capture ditolak: di Linux, pastikan user sudah di group wireshark dan relogin.
  • Wi-Fi monitor mode: dukungan terbatas di Windows; pertimbangkan Linux + adapter kompatibel untuk kebutuhan advanced.
