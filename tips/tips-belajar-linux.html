  • Mulai dari distribusi ramah pemula: Ubuntu, Linux Mint, atau Fedora.
  • Biasakan terminal: gunakan command line setiap hari.
  • Pelajari struktur file system: direktori /home, /etc, /var.

  • Praktik nyata: install software, buat user, edit file konfigurasi.
